1. What is Box sizing ?

  • the CSS box-sizing allows you to include padding and border in an elements total width and height.
  • if you set box-sizing : border-box; the padding and border are included in the width and height

2. What is animation delay ?

  • The animation-delay property in CSS, specifies the amount of time to wait before applying the
    animation to the elemnts before beginning the effect.

3. Which property to use to merge the cells vertically in grid ?

In CSS to merge the cells vertically we use the propert Grid-column.
The syntax to merge cells in vertical manner we use grid-column: (from where to start) / (where to end)

4. Which property to use to merge cells horizontally in grid ?

In CSS to merge the cells horizontally we use the propert Grid-column.
The syntax to merge cells in horizontal manner we use grid-row: (from where to start) / (where to end)
